# 八抓鱼
这是一款探针采集程序

### 架构设计
![avatar](/doc/img/1.png)

### 特点
- 探针可以使用任何编程语言编写，输出都按标准输出即可
- 动态监测探针的变化，自动更新探针与发现探针

### 使用说明
- 编写采集探针放到 collectors 目录下面
  - **0** 表示0S运行一次，也就是这个探针任务会一直运行下去
  - **3** 表示3秒运行一次
  - 其余以此类推

探针中只需要按照标准输出"print"输出就可以

- middlewares 中间件存放的是一些发送端的中间件，这部分仓考编写的案列代码
